혼자 만드는 게임, CLAUDE가 팀이 된다

기획·코드·아트 방향·밸런싱·마케팅까지 1인이 다 해야 하는 인디게임 개발. Claude 플러그인을 조합하면 Claude가 기획자·QA·마케터 역할을 동시에 수행해줘요. 아래는 2D 사이드스크롤 액션게임 개발 실전 시나리오입니다.

01
전투 밸런스 설계
→ PRODUCTIVITY + ENGINEERING
"플레이어 공격력·체력·적 패턴을 어떻게 수치로 잡아야 할지 모르겠어..."
Claude 프롬프트 예시

"2D 사이드스크롤 액션게임이야. 플레이어 HP 100 기준으로 일반 몹·엘리트·보스의 체력·공격력·넉백 수치 표를 만들어줘. 3스테이지 진행형이고 체감 난이도가 완만하게 올라야 해."

Claude가 해줘요
›스테이지별 수치 밸런스 표 (HP·ATK·속도·보상 골드)
›플레이어 예상 생존 시간 계산 공식 제공
›밸런스 조정 시 수치 재계산 즉시 가능
02
플레이어 이동 코드 디버깅
→ ENGINEERING
"점프 중에 벽 타기 판정이 이상하게 겹쳐. Unity C# 코드가 뭔가 잘못됐는데..."
Claude 프롬프트 예시

"Unity 2D 플레이어 컨트롤러야. 점프 중 WallSlide 상태 진입 조건이 이상해서 벽에 닿지 않아도 슬라이딩이 트리거돼. 코드 봐줘: [코드 붙여넣기]"

Claude가 해줘요
›버그 원인 지목 + 수정 코드 즉시 제공
›상태머신 구조 개선 제안 (Idle·Run·Jump·WallSlide)
›엣지케이스 체크리스트 추가 제공
03
스테이지 레벨 디자인 문서화
→ PRODUCTIVITY + ENGINEERING
"아이디어는 많은데 어디에 무엇을 배치할지 정리가 안 돼. 레벨 디자인 문서도 없고..."
Claude 프롬프트 예시

"2D 사이드스크롤 액션 1-1 스테이지 레벨 디자인 문서 만들어줘. 테마: 폐공장. 등장 적: 드론·가드로봇. 길이: 3분. 마크다운 표로 구역별 구성 정리해줘."

Claude가 해줘요
›구역별 레이아웃·적 배치·장치 설명 문서 생성
›플레이어 동선 흐름 분석 및 난이도 피드백
›이후 스테이지 연계 설계 힌트 제공
04
Steam 출시 마케팅 문구
→ MARKETING + BRAND VOICE
"Steam 페이지 설명 쓰기가 너무 어려워. 내 게임의 매력을 어떻게 표현해야 할지..."
Claude 프롬프트 예시

"2D 사이드스크롤 액션. 주인공 사이보그 용병, 디스토피아 미래 도시 배경, 빠른 콤보와 파쿠르 핵심. Steam 스토어 짧은 설명(150자)과 긴 설명(500자) 한국어·영어 둘 다 써줘."

Claude가 해줘요
›Steam 최적화된 숏·롱 설명문 한/영 4가지 버전
›핵심 키워드 강조 + 구매 유도 CTA 포함
›태그 추천·경쟁작 포지셔닝 전략까지 제공
05
플레이테스트 피드백 분석
→ DATA + PRODUCTIVITY
"베타 테스터 10명 설문 결과가 있는데, 의견이 제각각이라 뭘 먼저 고쳐야 할지 모르겠어..."
Claude 프롬프트 예시

"플레이테스트 설문 결과야. [텍스트 붙여넣기] 공통 불만 패턴 찾아서 우선순위 높은 순으로 개선 항목 정리해줘. 개발 공수 적은 것부터 먼저 고칠 수 있게 구분도 해줘."

Claude가 해줘요
›피드백 공통 패턴 분류 (UX·밸런스·버그·요청)
›개발 공수 vs 개선 효과 매트릭스 제공
›다음 스프린트 TO-DO 리스트 자동 생성
06
게임 세계관·스토리 집필
→ PRODUCTIVITY + BRAND VOICE
"세계관 설정은 머릿속에 있는데, 플레이어에게 자연스럽게 전달할 방법을 모르겠어..."
Claude 프롬프트 예시

"디스토피아 2087년 배경. 주인공은 기억을 잃은 사이보그 용병 KAIN. 1스테이지 클리어 후 컷씬 대사 3줄 써줘. 분위기: 쓸쓸하고 결의에 찬 느낌."

Claude가 해줘요
›분위기에 맞는 컷씬 대사·나레이션 즉시 생성
›캐릭터 말투 일관성 유지 및 로어북 확장 가능
›한국어·영어 현지화 대사 동시 작성 가능
